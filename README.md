# Sinflix Modifier

A userscript that enhances the SinFlix drama index and its associated download pages — adding quick search, Google & MyDramaList lookups, and more.

**Author:** [mthlpbs](https://greasyfork.org/en/users/1490967-mthlpbs) · **License:** MIT

---

## Installation

1. Install a userscript manager for your browser:
   - [Violentmonkey](https://violentmonkey.github.io/) *(recommended)*
   - [Tampermonkey](https://www.tampermonkey.net/)

2. Click the link below to install the script — your userscript manager will prompt you to confirm:

   **[Install Sinflix Modifier](https://raw.githubusercontent.com/mthlpbs/sinflix-modifier/refs/heads/main/sinflix-modifier.user.js)**

That's it. Visit any of the supported pages and the script will activate automatically.
